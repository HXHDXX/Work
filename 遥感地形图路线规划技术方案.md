# 遥感地形图生成与路线规划技术方案

## 一、遥感影像生成地形图

### 技术原理
利用**立体视觉**或**雷达干涉**原理：从不同角度获取同一地表的两幅影像，通过三角测量计算像素的三维坐标，重建地形表面。

| 步骤 | 技术原理 | 输出 |
|------|---------|------|
| 1. 获取影像 | 光学立体成像（交会角形成视差）或InSAR（雷达相位差） | 原始影像对 |
| 2. 立体匹配/干涉处理 | **摄影测量前方交会**或**干涉相位解缠**：通过像点坐标差反算地面高程 | 点云数据 |
| 3. 滤波去噪 | **形态学滤波+布料模拟**：区分地面点与非地面点（植被、建筑），剔除非地面高程 | 裸地高程点 |
| 4. 插值成栅格 | **反距离加权或克里金插值**：将离散点云内插为连续规则网格 | DEM栅格图 |

**精度**：卫星方法约2–10米，航空/无人机方法可达0.1–1米。

## 二、基于地形图生成路线

### 技术原理
将DEM视为**三维代价场**，路径规划转化为**带约束的最优化问题**：在满足安全阈值的前提下，寻找累积代价最小的连续路径。

### 直升机飞行路线

```
输入：DEM + 起点/终点
  ↓
1. 沿预定走廊提取高程剖面
   → 原理：DEM栅格双线性插值，生成航线正下方地形断面

2. 计算每段最低安全高度 = 地形高程 + 越障余度（通常100–300米）
   → 原理：安全包络面建模，确保任何点满足MSA（最低安全高度）约束

3. 标记超限区（陡坡、窄谷、超高障碍物）
   → 原理：地形坡度>阈值或通道宽度<<旋翼直径×2时，节点代价设为无穷大

4. 用路径算法（A*）搜索满足高度约束的最短路径
   → 原理：启发式图搜索，代价函数 = 实际航段距离 + 预估剩余距离，优先扩展低代价节点

5. 预设紧急着陆区（坡度<<5°、面积>30×30米的平坦地）
   → 原理：局部窗口坡度分析+区域生长法，提取符合尺寸和坡度阈值的连通平坦区
  ↓
输出：航迹点序列 + 高程剖面 + 危险点标注
```

### 越野车行驶路线

```
输入：DEM + 起点/终点
  ↓
1. 计算坡度图（提取>15°陡坡、>25°禁行区）
   → 原理：3×3窗口高程差分，arctan(高程差/水平距)得坡度角

2. 计算地形粗糙度（识别岩石/沟壑区）
   → 原理：局部高程标准差或残差分析，粗糙度大=通行性低

3. 提取汇水线（避开冲沟、潜在洪水区）
   → 原理：D8单流向算法，模拟水流累积量，识别河谷网络

4. 通行性评分 = 坡度 + 粗糙度 + 距水系距离
   → 原理：加权线性组合，各因子归一化后按权重叠加为综合代价栅格

5. 搜索最低代价路径
   → 原理：Dijkstra或A*算法，在加权栅格中寻找起点到终点的最小累积代价路径

6. 标注涉水点、孤立路段、救援困难区
   → 原理：路径与河网交集检测+缓冲区分析，孤立度=距最近道路网距离
  ↓
输出：推荐路线 + 里程/耗时估算 + 风险提示
```

## 三、关键局限

| 环节 | 局限 | 应对措施 |
|------|------|---------|
| DEM精度 | 卫星DEM含植被高，低空飞行/越野时误差致命 | 植被区用LiDAR或实地测量修正 |
| 动态障碍物 | 地形图不含电线、施工塔吊、新冲沟 | 叠加光学影像目视判读，关键区无人机侦察 |
| 地表性质 | DEM无法分辨沙地/泥地/岩石的通过性 | 结合卫星影像纹理分类，或现场验证 |

---

## 四、结论

- **高空巡航/常规越野**：卫星DEM直接可用，流程闭环可行。
- **低空贴地/极限越野**：DEM精度不足，必须引入无人机LiDAR或机载实时传感器，否则风险不可控。
